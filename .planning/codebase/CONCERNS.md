# Technical Concerns

**Analysis Date:** 2026-03-22

## Tech Debt

| Area | Issue | Location |
|------|-------|----------|
| Dependencies | `tapi` pinned to floating git HEAD (no commit SHA) — silent breaking changes possible | `Cargo.toml` |
| Driver | `run.toml` commands split on spaces — breaks paths containing spaces | `crates/driver/` |
| Hub memory | Jobs accumulate in `Hub` indefinitely — commented-out cleanup limit | `crates/driver/src/hub.rs` |
| Scoreboard | Sends full state every 200ms with no change detection — unnecessary traffic | `crates/inspectify/src/checko/scoreboard.rs` |

## Known Bugs

| Severity | Bug | Location |
|----------|-----|----------|
| Medium | Double `RwLock` acquisition in `Hub::jobs()` — TOCTOU race condition | `crates/driver/src/hub.rs` |
| High | `lines().last().unwrap()` panics on empty string | `crates/chip-wasm/src/lib.rs` |
| Medium | `GroupStatus::Finished` set after each test input, not just the final one | `crates/inspectify/src/checko.rs` |
| High | `checko.work()` unwrapped in a spawned task — unrecoverable error crashes server | `crates/inspectify/src/checko.rs` |

## Security Concerns

| Risk | Detail |
|------|--------|
| ⚠️ Open CORS | `CorsLayer::permissive()` on all API routes — no origin restrictions |
| ⚠️ No authentication | All endpoints (`/generate`, `/exec_analysis`, `/checko/csv`, etc.) are unauthenticated |
| ⚠️ Unsandboxed execution | Student `run.toml` commands executed directly as OS processes — no resource limits or sandboxing |

These are acceptable for the intended use case (local dev tool / controlled classroom grading server), but would be critical issues if exposed publicly.

## Performance

| Area | Issue | Impact |
|------|-------|--------|
| Job lookup | `O(n)` linear scan in `Hub::get_job()` by `JobId` | Degrades with many jobs |
| SQLite | Single `std::sync::Mutex` serializes all DB access in `checko/db.rs` | Bottleneck under concurrent group runs |
| Scoreboard | Full recompute + broadcast every 200ms regardless of changes | Unnecessary CPU + bandwidth |

## Areas to Watch When Modifying

- **`crates/ce-shell/src/def.rs`** — `define_shell!` macro is complex; changes affect all analysis dispatch
- **`apps/inspectify/src/lib/api.ts`** — auto-generated by `tapi`; manual edits will be overwritten on server restart
- **`crates/driver/src/hub.rs`** — concurrent access patterns; RwLock usage needs care
- **`run.toml` parsing** — space-splitting of commands is a known fragility
