<objective>
Add co-generated witness memories to the compiler test generation pipeline.

Purpose: The current oracle uses a fixed-seed RNG to produce exactly 10 memory states for every program. Guards whose conditions are never satisfied by those 10 states go unvalidated. Co-generating witness memories per-program guarantees every guard is exercised in both directions.

Output: Updated `Input` struct with `witness_mems: Vec<InterpreterMemory>`, witness generation logic in `gcl_compiler_gen.rs`, and `validate()` consuming witnesses instead of fixed-seed samples.
</objective>

1. `pub fn collect_guards(cmds: &Commands) -> Vec<BExpr>` — Recursively walk the `Commands` AST. For each `Command::If(guards)` and `Command::Loop(guards)`, push `b.clone()` from each `Guard(b, body)` into the result, then recurse into `body`. Skip `Command::Assignment` and `Command::Skip`. 2. `pub fn generate_witness_memories<R: Rng>(commands: &Commands, rng: &mut R) -> Vec<InterpreterMemory>` —
   - Call `collect_guards(commands)` to get all guard BExprs.
   - Compute `fv = commands.fv()` once.
   - For each guard BExpr, attempt up to 100 random memories (using `Memory::from_targets_with(fv.clone(), rng, ...)` with `rng.random_range(-10..=10)` for scalars, `rng.random_range(5..=10)` length arrays with `rng.random_range(-10..=10)` elements — same pattern as existing oracle).
   - For each candidate memory, evaluate `bexpr.semantics(&mem)`. If `Ok(true)`, store as true-witness. If `Ok(false)`, store as false-witness. Stop early for that direction once found.
   - Collect all found witnesses into a single `Vec<InterpreterMemory>` (deduplicating is not necessary — duplicates are harmless).
   - If a guard is a tautology (`BExpr::Bool(true)`) or contradiction (`BExpr::Bool(false)`), the unsatisfiable direction simply won't find a witness — that's fine, skip it.
   - Add necessary imports at the top: `use gcl::interpreter::InterpreterMemory;`, `use gcl::memory::Memory;`, `use gcl::ast::BExpr;` (BExpr is already imported, verify).

```rust
#[test]
fn witness_memories_cover_all_guards() {
    use gcl::ast::Commands;
    use rand::SeedableRng;
    use ce_core::gn::compiler_gen::{collect_guards, generate_witness_memories, CompilerContext, gen_commands};

    let mut rng = rand::rngs::SmallRng::seed_from_u64(42);
    // Generate 20 random programs and verify witnesses cover guards
    for _ in 0..20 {
        let commands = gen_commands(&mut CompilerContext::default(), &mut rng);
        let guards = collect_guards(&commands);
        let witnesses = generate_witness_memories(&commands, &mut rng);

        // For every non-tautology/non-contradiction guard, at least one witness
        // should evaluate that guard (either true or false)
        for guard in &guards {
            match guard {
                gcl::ast::BExpr::Bool(_) => continue, // skip trivial guards
                _ => {
                    let any_evaluates = witnesses.iter().any(|mem| {
                        guard.semantics(mem).is_ok()
                    });
                    assert!(any_evaluates, "No witness evaluates guard: {:?}", guard);
                }
            }
        }
    }
}
```
